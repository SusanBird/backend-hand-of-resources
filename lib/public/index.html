Hand of Resources

Route Method Description Example

Dogs

 
/dogs
 GET Gets a list of all dogs. List of Dogs 
/dogs/1
 GET Get a certain dog's info. List One Dog 
/dogs/
 POST Adds a dog. curl -d '{"name": "French", "age": "4", "iq": "114", "energy": "low"}' -H 'Content-Type: application/json' 'https://backend-hand-of-resources2.herokuapp.com/dogs' 
/dogs/3
 PUT Updates a dog. curl -X PUT -H "Content-Type: application/json" -d '{"age":"3"}' https://backend-hand-of-resources2.herokuapp.com/dogs/3 
/dogs/1
 DELETE Deletes a dog. curl -X DELETE https://backend-hand-of-resources2.herokuapp.com/dogs/1 -H "Accept: application/json"
